# 🤖 Bunny AI Chatbot

A simple rule-based AI chatbot built to respond to user queries using predefined responses. This project was created as **Task 1** of my **AI Internship at CodeSoft**.

## 🚀 Features
- Responds to greetings (Hi, Hello, Good Morning, Good Night)
- Answers basic questions (Name, AI, Python, etc.)
- Helps with homework and coding-related queries
- Handles unknown inputs with a default response
- Simple, fast, and beginner-friendly

## 🛠️ Technologies Used
- React
- TypeScript
- Tailwind CSS
- Vite

## ▶️ How to Run
```bash
npm install
npm run dev
```
